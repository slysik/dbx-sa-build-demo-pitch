# dbx-sa-build-demo-pitch

A Databricks Solutions Architect build demo pitch repository showcasing key Databricks platform capabilities and use cases.

## Overview

This repository serves as a demonstration resource for Databricks Solutions Architects, providing example code, notebooks, and configuration to illustrate the value of the Databricks Lakehouse Platform.

## Contents

- **Notebooks** – Sample Databricks notebooks demonstrating data engineering, machine learning, and analytics workflows
- **Configuration** – Infrastructure-as-code and cluster configuration examples
- **Pipelines** – Delta Live Tables (DLT) pipeline definitions
- **Documentation** – Supporting materials for demo presentations

## Getting Started

### Prerequisites

- A Databricks workspace (Community Edition, or full workspace)
- [Databricks CLI](https://docs.databricks.com/dev-tools/cli/index.html) installed and configured
- Python 3.8+

### Setup

1. Clone this repository:
   ```bash
   git clone https://github.com/slysik/dbx-sa-build-demo-pitch.git
   cd dbx-sa-build-demo-pitch
   ```

2. Configure the Databricks CLI with your workspace:
   ```bash
   databricks configure --token
   ```

3. Import the notebooks into your Databricks workspace or follow the instructions in each subdirectory.

## Usage

Refer to the individual directories and notebooks for specific instructions. Each demo section is self-contained and includes inline documentation.
